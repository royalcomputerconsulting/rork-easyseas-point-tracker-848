# Financials Data Directory

This directory stores financial data for the cruise tracking application.

## Files:
- `financials.csv` - Main financial data export
- `snapshots/` - Historical snapshots of financial data

## Data Sources:
- Receipt data from OCR processing
- Statement data from cruise statements
- Manual entries and corrections

## Note:
The persist.json file is stored in /DATA/persist.json for all application data persistence.
